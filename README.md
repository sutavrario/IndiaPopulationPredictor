# 🇮🇳 India Population Prediction System

## Overview

The India Population Prediction System is a Machine Learning-based web application that predicts India's future population using historical population data. The model is trained using Python and Scikit-Learn in Google Colab and deployed through a Flask-based web application.

Users can enter any future year through the web interface and receive an estimated population prediction instantly.

---

## Features

* Predict future population of India
* Machine Learning-based forecasting

## How It Works

1. Historical population data is collected and processed.
2. A Machine Learning model is trained using population records.
3. The trained model is saved and integrated with a Flask backend.
4. Users enter a future year through the web interface.
5. The system predicts and displays the estimated population for that year.
